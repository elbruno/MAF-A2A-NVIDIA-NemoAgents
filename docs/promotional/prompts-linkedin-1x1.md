# LinkedIn Image Prompts (1:1)

Use this command format:

```bash
t2i "<prompt>" --provider foundry-mai2 --width 1024 --height 1024 --out "docs/promotional/images/linkedin-square-1024x1024.png"
```

## Prompt A (recommended)

```text
Modern square tech illustration for LinkedIn (1:1). Show a multi-agent AI system with three clear blocks: "NeMo Data Analysis Agent" on the left with charts and anomaly markers, "Web Chat Interface" in the center, and "MAF Action Agent" on the right with alert/report/action icons. Add directional arrows for A2A flow: Analyze -> Decide -> Act. Include subtle Azure Aspire dashboard motifs in the background. Professional enterprise style, clean typography, blue and purple palette, high contrast, no logo infringements, no watermarks.
```

Square promotional hero image (1:1) for an open-source AI project.
