Building multi-agent systems is easier when each agent is specialized.

In this open-source sample, we combine:
- NVIDIA NeMo Agent Toolkit for data analysis
- Microsoft Agent Framework for action execution
- A2A communication for interoperability
- Aspire orchestration for observability and reliability

Result: analyze trends and trigger actions in one cohesive workflow.
