# Repository Social Prompts

Use this command format:

```bash
t2i "<prompt>" --provider foundry-mai2 --width 1200 --height 630 --out "docs/promotional/images/repo-social-1200x630.png"
```

> 1200x630 works well for GitHub/social preview cards, X, and most link-sharing platforms.

## Prompt A (recommended)

```text
Social preview card image (1200x630) for open-source repository "MAF-A2A-NVIDIA-NemoAgents". Show three labeled modules: NeMo Analysis Agent, Web Chat Interface, and MAF Action Agent, connected by clean arrows and A2A protocol indicators. Include concise visual cues for "Analyze", "Route", "Act". Modern open-source developer aesthetic, crisp UI-inspired layout, strong readability in small previews, no watermark.
```

## Prompt B (alternative)

```text
Professional software project social card in 1200x630 format. Highlight multi-agent collaboration and orchestration: charts and anomaly signals on left, orchestration/chat in center, actionable alerts and reports on right. Include subtle references to Aspire orchestration and observability traces. Minimalist but bold, high contrast, suitable for GitHub repository social sharing.
```
